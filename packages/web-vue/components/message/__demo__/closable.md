## zh-CN

设置 `closable` 来显示关闭按钮。

---

## en-US

Set `closable` to show the close button.

---

```vue
<template>
  <a-button @click="handleClick">Closable Message</a-button>
</template>

<script>
export default {
  methods: {
    handleClick(){
      this.$message.info({
        content:'This is an info message!',
        closable: true
      })
    }
  }
};
</script>
```
