## zh-CN

通过设置 `marks` 可以添加文本标签。

---

## en-US

You can add text labels by setting `marks`.

---

```vue
<template>
  <a-slider :default-value="5" :style="{ width: '300px' }" :max="15" :marks="marks" />
</template>

<script setup>
const marks = {
  0: '0km',
  5: '5km',
  10: '10km',
  15: '15km',
};
</script>
```
