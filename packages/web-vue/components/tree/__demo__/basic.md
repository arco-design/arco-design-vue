## zh-CN

为每个节点赋予全局唯一的 `key`（必填项），`title` 为该节点显示的内容。

---

## en-US

Give each node a globally unique `key` (required), and the `title` is the content to be displayed on the node.

---

```vue
<template>
  <a-tree
    :data="treeData"
    :default-expanded-keys="['0-0-0']"
    :default-selected-keys="['0-0-0', '0-0-1']"
  />
</template>

<script setup>
import { ref } from 'vue';

const treeData = ref([
  {
    title: 'Trunk 0-0',
    key: '0-0',
    children: [
      {
        title: 'Branch 0-0-0',
        key: '0-0-0',
        disabled: true,
        children: [
          {
            title: 'Leaf',
            key: '0-0-0-0',
          },
          {
            title: 'Leaf',
            key: '0-0-0-1',
          }
        ]
      },
      {
        title: 'Branch 0-0-1',
        key: '0-0-1',
        children: [
          {
            title: 'Leaf',
            key: '0-0-1-0',
          },
        ]
      },
    ],
  },
]);
</script>
```
