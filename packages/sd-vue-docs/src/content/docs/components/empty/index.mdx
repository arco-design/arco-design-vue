## API

### `<empty>` Props

| 参数名             | 描述                         | 类型      | 默认值  | 版本   |
| ------------------ | ---------------------------- | --------- | :-----: | :----- |
| description        | 描述内容                     | `string`  |   `-`   |        |
| img-src            | 自定义图片的地址             | `string`  |   `-`   |        |
| in-config-provider | 是否在 ConfigProvider 中使用 | `boolean` | `false` | 2.47.0 |

### `<empty>` Slots

| 插槽名 |   描述    | 参数 |
| ------ | :-------: | ---- |
| image  | 图片/图标 | -    |
